# Claw

Generative UI client for OpenClaw. Agents respond with interactive UI (charts, cards, tables, forms) instead of markdown.

## Packages

| Package | Description |
|---------|-------------|
| [`packages/claw`](packages/claw) | `@openuidev/claw` — Static Next.js web client |
| [`packages/claw-plugin`](packages/claw-plugin) | `@openuidev/claw-plugin` — OpenClaw server-side plugin |

## Setup

```bash
pnpm install
```

## Development

```bash
cd packages/claw
pnpm dev
```

## Install Plugin

```bash
openclaw plugins install @openuidev/claw-plugin
```
